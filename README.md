# Mental-Health-Companion-Chatbot
**About**<BR>
The AI Mental Health Companion is an interactive tool that leverages the power of the Google Gemini API to offer empathetic and personalized support. It provides users with a safe space to express their feelings and receive constructive feedback inspired by Cognitive Behavioral Therapy (CBT) principles. The application includes a conversational interface for real-time support and a private diary for personal reflection.

**Features**<Br>
Personalized Profile: Create a user profile with details like nickname, age, and occupation to get tailored responses.<br>
AI-Powered Counselor: Interact with an AI that provides empathetic, mood, and motivational support. Responses include actionable tips and affirmations.<BR>
Daily Diary: A private, persistent space to write and save your thoughts and reflections.<BR>
